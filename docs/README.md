### 贡献指南

**添加新文档**:
1. 确定分类（01-07）
2. 使用标准模板
3. 添加到本索引
4. 提交PR review

**修改现有文档**:
1. Check out最新代码
2. 进行修改
3. 更新版本号
4. 记录变更日志

**文档命名规范**:
```
格式: NN-文档名称.md
示例: 
- 01-技术架构升级方案.md
- 02-商业运营方案.md
- 03-研发周期与里程碑.md
```

### 版本控制

```
文档版本格式: v主版本.次版本
- 主版本: 重大重构
- 次版本: 内容更新

示例:
v1.0 - 初始版本
v1.1 - 添加新章节
v2.0 - 重写主要内容
```
