## 🏛️ hello-agent 赛博小镇架构特性

### 核心组件

✅ **向量记忆系统 (LanceDB)**
- 384 维语义嵌入存储
- 智能路由到本地 Ollama 生成嵌入
- 元数据过滤（NPC、日期、情感、重要性）
- 语义相似度搜索（Top-K 检索）

✅ **MCP 协议适配器 (JSON-RPC 2.0)**
- 标准化 Agent-Tool 通信协议
- 工具注册和发现机制
- 5 个内置游戏工具：
  - `get_npc_info` - 获取 NPC 状态
  - `get_world_state` - 获取世界状态
  - `set_npc_mood` - 设置 NPC 心情
  - `trigger_event` - 触发游戏事件
  - `query_memory` - 查询记忆
- 参数验证和错误处理

✅ **记忆增强型对话**
```
Retrieve → Augment → Generate → Store
   ↓           ↓           ↓          ↓
语义搜索   注入上下文   LLM 生成   存储新记忆
```

✅ **工具注册机制**
- 动态工具注册/注销
- 工具元数据描述
- 自动文档生成
- Agent 自主调用能力

### 架构层次

```
┌─────────────────────────────────────┐
│     Godot Frontend (Game)           │
└──────────────┬──────────────────────┘
               │ HTTP/WebSocket
               ▼
┌─────────────────────────────────────┐
│     FastAPI Backend                 │
│  ┌───────────────────────────────┐  │
│  │  Memory + MCP Integration     │  │
│  └───────────────────────────────┘  │
└──────────────┬──────────────────────┘
               │ JSON-RPC 2.0
               ▼
┌─────────────────────────────────────┐
│     LLM Router                      │
│  ┌─────────┬─────────┬──────────┐  │
│  │ Ollama  │  Qwen   │ Gemini   │  │
│  └─────────┴─────────┴──────────┘  │
└──────────────┬──────────────────────┘
               │ Embeddings
               ▼
┌─────────────────────────────────────┐
│     LanceDB Vector Store            │
│     (NPC Long-term Memories)        │
└─────────────────────────────────────┘
```

### 设计模式

- ✅ **Repository Pattern** - 数据访问抽象
- ✅ **Strategy Pattern** - LLM Provider 切换
- ✅ **Registry Pattern** - 工具注册和管理
- ✅ **Observer Pattern** - 事件驱动架构
- ✅ **Adapter Pattern** - MCP 协议适配
