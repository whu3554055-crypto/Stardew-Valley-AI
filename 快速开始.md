# AI NPC 系统 - 快速开始指南

## 你的环境配置

✅ **Ollama**: 已安装  
✅ **模型**: qwen3.5:9b (6.6 GB)  
✅ **Godot**: v4.6.2  

## 快速启动（3步）

### 方法 1：一键启动（推荐）

双击运行 `start_game.bat`

这个脚本会自动：
1. 检查并启动 Ollama 服务
2. 验证 qwen3.5:9b 模型
3. 用 Godot 打开项目

### 方法 2：手动启动

```bash
# 1. 启动 Ollama（如果还没运行）
ollama serve

# 2. 用 Godot 打开项目
D:\program\Godot_v4.6.2-stable_win64.exe --path d:\repo\stardew_valley
```

## 在游戏中

### 测试 AI NPC

1. **按 F5 运行游戏**
2. 走到 NPC 旁边（Pierre、Abigail 或 Mayor Lewis）
3. 按 **E** 键与他们对话
4. NPC 会使用 AI 生成个性化回复！

### AI 配置

如果你想修改 AI 设置：

1. 在游戏里打开 AI 配置界面（需要添加到 UI）
2. 或者在代码中修改 `autoload/ai_agent_manager.gd`

当前配置：
- **模型**: qwen3.5:9b
- **API地址**: http://localhost:11434
- **温度**: 0.8（创意程度）
- **最大token**: 256（回复长度）

## AI 系统特性

### 1. 动态对话
NPC 会根据以下因素生成回复：
- 当前时间（早上/下午/晚上）
- 天气（晴天/雨天/雪天）
- 季节（春/夏/秋/冬）
- 与玩家的关系等级
- 过去的互动历史
- NPC 的心情状态

### 2. 记忆系统
- NPC 记得你们之前的对话
- 会提及过去的事件
- 关系等级会随互动提升
- 学习玩家的喜好

### 3. 情绪系统
- 15种基本情绪（开心、悲伤、兴奋、害羞等）
- 情绪会影响行为和对话
- 事件会触发情绪变化
- 情绪随时间自然衰减

### 4. 个性档案
每个 NPC 有独特的：
- 性格特质（友好度、耐心、能量等）
- 职业背景故事
- 说话风格
- 兴趣爱好

## 自定义 AI NPC

### 创建新的 AI NPC

1. 复制一个现有的 NPC 场景文件
2. 修改参数：

```gdscript
[node name="MyNPC" type="CharacterBody2D"]
npc_id = "my_unique_id"  # 唯一ID
npc_name = "角色名字"
use_ai_dialogue = true  # 启用AI对话

# 个性设定
ai_personality = {
    "traits": ["友好", "聪明"],  # 性格特质
    "occupation": "职业",
    "backstory": "背景故事...",
    "speech_style": "casual",  # 说话风格
    "interests": ["兴趣1", "兴趣2"]
}
```

3. 在主场景中添加这个 NPC

### 说话风格选项

- `"casual"` - 随意、友好
- `"formal"` - 正式、礼貌
- `"shy"` - 害羞、犹豫
- `"energetic"` - 充满活力
- `"mysterious"` - 神秘、cryptic
- `"gruff"` - 粗鲁、直接

## 故障排除

### AI 没有响应

1. **检查 Ollama 是否运行**
   ```bash
   ollama ps
   ```

2. **确认模型存在**
   ```bash
   ollama list | findstr qwen3.5
   ```

3. **查看 Godot 控制台输出**
   - 会有详细的错误信息

### 响应太慢

- qwen3.5:9b 是比较大的模型，首次响应可能需要几秒
- 后续对话会因为缓存而更快
- 可以切换到更小的模型（如果有）

### 想要更快的响应

编辑 `autoload/ai_agent_manager.gd`：
```gdscript
var api_config = {
    "model": "qwen3.5:9b",
    "max_tokens": 128,  # 减小这个值（默认256）
    "temperature": 0.7  # 降低创意度会更快
}
```

