### Milestone 1: 技术基础重构

**时间**: 2026-04-01 ~ 2026-04-30 (4周)  
**负责人**: 主程  
**目标**: 建立稳定的数据层和架构基础

#### Week 1: SQLite集成

**任务**:
- [ ] Day 1-2: 集成godot-sqlite GDExtension
  - 下载预编译二进制
  - 配置项目依赖
  - 编写测试用例
- [ ] Day 3-4: 设计数据库Schema
  - NPC记忆表
  - 关系表
  - 玩家进度表
  - 物品库存表
- [ ] Day 5: 实现DatabaseManager单例
  - 连接管理
  - CRUD操作封装
  - 事务支持

**验收标准**:
```gdscript
# 可以执行以下操作
var db = DatabaseManager.get_instance()
db.begin_transaction()
db.insert("npc_memories", {...})
db.commit()  # 或 rollback()
```

**应对**: 准备备用方案（保持JSON + WAL日志）
